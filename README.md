# Watcher

Watcher is a Discord moderation bot built to help manage servers by handling commands, filtering content, tracking warnings, and notifying moderators through an in-Discord UI.

## Features

- Slash/prefix commands for moderation actions
- Message filtering (e.g. banned words, spam, links)
- Warning system to track user infractions
- Interactive moderation menu (UI-based actions instead of typing commands)
- Notification system for alerting moderators
- Event-driven architecture for reacting to Discord events (joins, messages, etc.)

## Setup

1. Clone the repository
   ```
   git clone https://github.com/karenfaithconde/watcher-bot.git
   ```
2. Install dependencies
   ```
   pip install -r requirements.txt
   ```
3. Configure your bot token and settings in `config.py`
4. Run the bot
   ```
   python main.py
   ```
